# Start Frontend - Quick Guide

## ✅ Backend Status
**Backend is RUNNING** on http://localhost:8081

## 🚀 Start Frontend Now

### Step 1: Open New Terminal
Open a **new terminal window** (don't close this one)

### Step 2: Navigate to Frontend Folder
```bash
cd ..\auth-frontend
```

### Step 3: Start Frontend
```bash
npm start
```

The frontend will:
- Start on http://localhost:3000
- Automatically open in your browser
- Connect to backend at http://localhost:8081

## ⏱️ Wait Time
Frontend takes about 10-20 seconds to start.

## 🌐 Once Started

You'll see the login page at: http://localhost:3000
